Historia del HTML

Tím Berners-Lee

HTML fue creado en el año de 1991 por Tím Berners-Lee en la Organización Europea para la Investigación Nuclear (CERN). Su objetivo era facilitar el intercambio de documentos científicos a través de una red global, lo que llevó al desarrollo del World Wide Web.

Su primera versión el HTML 1.0, era muy básica y solo incluía alrededor de 20 etiquetas para estructurar texto, enlaces e imágenes.

Evolución principal a lo largo del tiempo

  • HTML 1.0 (1991): Primera versión
  • HTML 2.0 (1995): Introdujo formularios y tablas.
  • HTML 3.2 (1997): Añadió soporte para scripts y applets.
  • HTML 4.01 (1999): Mejoró la accesibilidad y separó el contenido de la presentación.
  • XHTML 1.0 (2000): Reformuló HTML como una aplicación XML.
  • HTML5 (2014): Introdujo nuevas etiquetas semánticas, multimedia y APIs para mejorar la interactividad y funcionalidad web.
