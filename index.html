Bienvenidos al sitio de la Evolución del HTML

Introducción al HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear tus paginas web y esta nos describe una estructura básica de una pagina web mediante el uso de etiquetas.

Objetivo de nuestro sitio Web

Este sitio es creado con el objetivo de demostrar por la evolución que a pasado HTML desde su creación a la actualidad, describiendo los cambios mas significativos que a tenido.
